# 回文数


<extoc></extoc>

## 一、题目

[回文数](https://leetcode-cn.com/problems/palindrome-number/)

## 二、题解

```go
func isPalindrome(x int) bool {
    if x< 0{
        return false
    }
    s := fmt.Sprintf("%d", x)
    lenS := len(s)

    for i:= 0; i< lenS/2; i++{
        if s[i] != s[lenS-1-i]{
            return false
        }
    }
    return true
}
```
