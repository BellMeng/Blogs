# 6. Z字形变换

## 一、题目

[Z字形变换](https://leetcode-cn.com/problems/zigzag-conversion/)

## 二、题解

```go
func convert(s string, numRows int) string {
	zArr := make([]string, numRows)
	lenS := len(s)
	if numRows == 1{
		return s
	}
	row := 0
	down := true
	for i:= 0; i < lenS; i++{
		zArr[row] += fmt.Sprintf("%c", s[i])
		if down {
			row++
		}else{
			row--
		}
		if row == numRows -1||row == 0{
			down = !down
		}
	}
	s = ""
	for _, value := range zArr{
		s+=value
	}
	return s
}
```
