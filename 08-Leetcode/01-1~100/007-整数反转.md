# 整数反转


<extoc></extoc>

## 一、题目

[整数反转](https://leetcode-cn.com/problems/reverse-integer/)

## 二、题解

```go
func reverse(x int) int {
    res := 0
    for x != 0 {
        n := x % 10
        x /= 10
        tmp:= res*10 + n
        if tmp < math.MinInt32 || tmp > math.MaxInt32{
            return 0
        }
        res = tmp
    }
    return res
}
```
