# 罗马数字转整数


<extoc></extoc>

## 一、题目

[罗马数字转整数](https://leetcode-cn.com/problems/roman-to-integer/)

## 二、题解

```go
var symbolValues = map[byte]int{
    'I': 1,
    'V': 5,
    'X': 10,
    'L': 50,
    'C': 100,
    'D': 500,
    'M': 1000,
}

func romanToInt(s string) int {
    var result int = 0
    lengthS := len(s)
    for i := 0; i < lengthS;i++{
        if i < lengthS - 1 && symbolValues[s[i]] < symbolValues[s[i+1]]{
            result -= symbolValues[s[i]]
        }else {
            result += symbolValues[s[i]]
        }
    }
    return result
}
```
